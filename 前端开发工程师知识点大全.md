## 一、HTML / CSS

### 1.1 HTML 基础

- 文档结构 (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)
- 语义化标签 (`<header>`, `<nav>`, `<section>`, `<article>`, `<footer>`, `<main>`)
- 文本标签 (`<h1>-<h6>`, `<p>`, `<span>`, `<div>`, `<a>`, `<strong>`, `<em>`)
- 列表 (`<ul>`, `<ol>`, `<li>`, `<dl>`, `<dt>`, `<dd>`)
- 表格 (`<table>`, `<thead>`, `<tbody>`, `<tr>`, `<td>`, `<th>`, `<caption>`)
- 表单 (`<form>`, `<input>`, `<select>`, `<textarea>`, `<button>`, `<label>`)
- 多媒体 (`<img>`, `<video>`, `<audio>`, `<canvas>`, `<svg>`)
- DOM 结构与节点操作
- HTML5 新特性（LocalStorage, SessionStorage, Web Worker, WebSocket, Geolocation, Drag & Drop）

### 1.2 CSS 基础

- 选择器（元素、类、ID、属性、伪类、伪元素）
- 优先级与层叠（!important、 specificity、继承）
- 盒模型（content, padding, border, margin, box-sizing）
- 常见属性（display, position, float, overflow, visibility, opacity）
- 文本与字体（font-family, font-size, font-weight, text-align, line-height, color）
- 背景与边框（background, border, border-radius, box-shadow）
- Flexbox 布局
- Grid 布局
- 响应式布局（媒体查询 `@media`、rem、em、vw/vh）

### 1.3 CSS 进阶

- CSS 变量（自定义属性）
- 预处理器（Sass/SCSS, Less, PostCSS）
- CSS 模块化（CSS Modules, Styled Components）
- 动画与过渡（transition, animation, transform, keyframes）
- 层叠上下文与 z-index
- BFC（块级格式化上下文）与清除浮动
- CSS 性能优化（will-change, contain, GPU 加速）
- 移动端适配（viewport, 1px 边框问题,刘海屏适配）

### 1.4 CSS 框架

- Tailwind CSS
- Bootstrap
- Ant Design (Vue/React)
- Element UI / Plus (Vue)
- Material UI (React)

### 9.1 前端性能指标

- LCP (Largest Contentful Paint)
- FCP (First Contentful Paint)

- FID / INP (Interaction to Next Paint)
- CLS (Cumulative Layout Shift)

### 9.2 优化手段

- 减少请求次数（合并资源、懒加载）
- 减少资源体积（压缩、Tree Shaking、代码分割、图片优化、WebP）
- 减少渲染阻塞（CSS 放在头部、JS 放在底部或异步）
- CDN 加速
- 浏览器缓存（强缓存、协商缓存、Service Worker、PWA）

- CSS 优化（will-change, contain）
