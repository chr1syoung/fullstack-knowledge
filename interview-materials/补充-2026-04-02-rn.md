## 知识点总览

本次补充覆盖 React Native 全流程知识点：

### 开发基础
- RN 核心原理（JSX → 原生组件映射，Yoga 布局引擎）
- RN vs React Web 核心差异（组件库、布局、事件系统）
- 新架构三大组件：Fabric 同步渲染 / TurboModules 懒加载 / Hermes AOT 字节码

### 样式与布局
- Flexbox 与 CSS 差异（flexDirection 默认值、zIndex）
- 响应式适配（useWindowDimensions / PixelRatio / SafeAreaView）
- 样式复用（StyleSheet.create）

### 导航与状态
- React Navigation 5.x（Native Stack + Bottom Tabs）
- 页面参数传递（navigate / useRoute）
- Redux Toolkit（createSlice / createAsyncThunk）
- Zustand（轻量替代 Redux）

### 原生能力
- iOS 原生模块（Swift + Objective-C 桥接）
- Android 原生模块（Kotlin + ReactPackage 注册）
- TurboModules TypeScript 定义

### 打包发布
- iOS：JS Bundle 生成 → Xcode Archive → .ipa → App Store Connect
- Android：签名密钥 → JS Bundle → Gradle → .aab → Google Play
- AAB vs APK 区别

### CI/CD 与热更新
- GitHub Actions 全自动构建（iOS + Android）
- CodePush 热更新原理与发布命令
- 应用商店密钥管理（secrets）

### 性能优化
- FlatList 关键配置（removeClippedSubviews / windowSize）
- React.memo + useCallback 避免重渲染
- Hermes 启动优化 + Sentry 监控

### 工程化
- monorepo（pnpm workspaces + Turbo）
- TypeScript 路径别名
- 环境配置（dev / staging / prod）
- Sentry 错误监控
