## 十三、软技能

### 13.1 代码审查（Code Review）

**Code Review 的目的：**

1. 发现 bug 和潜在问题
2. 确保代码质量和一致性
3. 知识共享和团队学习
4. 维护代码库的可维护性

**Code Review 清单：**

```
功能性：
□ 代码是否实现了需求？
□ 边界条件是否处理？
□ 错误处理是否完善？

代码质量：
□ 代码是否易于理解？
□ 变量和函数命名是否清晰？
□ 是否有重复代码？
□ 函数是否单一职责？

性能：
□ 是否有不必要的循环或计算？
□ 是否有内存泄漏风险？
□ 数据库查询是否优化？

安全：
□ 是否有 XSS/CSRF 风险？
□ 敏感数据是否正确处理？
□ 权限验证是否完善？

测试：
□ 是否有单元测试？
□ 测试覆盖率是否足够？
```

### 13.2 微前端

**微前端概念：**

将大型前端应用拆分为多个独立的小应用，每个应用可以独立开发、部署。

**qiankun 使用：**

```javascript
// 主应用
import { registerMicroApps, start } from 'qiankun';

registerMicroApps([
    {
        name: 'react-app',
        entry: '//localhost:7100',
        container: '#container',
        activeRule: '/react',
        props: {
            name: 'qiankun',
        }
    },
    {
        name: 'vue-app',
        entry: '//localhost:7200',
        container: '#container',
        activeRule: '/vue'
    }
]);

start();

// 子应用（React）
export function bootstrap() {
    console.log('react app bootstraped');
}

export function mount(props) {
    ReactDOM.render(<App />, props.container.querySelector('#root'));
}

export function unmount(props) {
    ReactDOM.unmountComponentAtNode(props.container.querySelector('#root'));
}
```

### 13.3 性能监控

```javascript
// Sentry 错误监控
import * as Sentry from '@sentry/react';

Sentry.init({
    dsn: 'https://xxx@sentry.io/xxx',
    environment: process.env.NODE_ENV,
    tracesSampleRate: 1.0
});

// 自定义错误上报
Sentry.captureException(new Error('Something went wrong'));
Sentry.captureMessage('Something happened');

// 用户信息
Sentry.setUser({ id: '123', email: 'user@example.com' });

// 自定义性能监控
class PerformanceMonitor {
    static report(metric) {
        fetch('/api/metrics', {
            method: 'POST',
            body: JSON.stringify(metric),
            headers: { 'Content-Type': 'application/json' }
        });
    }
    
    static init() {
        // 监控 LCP
        new PerformanceObserver((list) => {
            const entries = list.getEntries();
            const lcp = entries[entries.length - 1];
            this.report({ type: 'LCP', value: lcp.startTime });
        }).observe({ type: 'largest-contentful-paint', buffered: true });
        
        // 监控 FID
        new PerformanceObserver((list) => {
            for (const entry of list.getEntries()) {
                this.report({
                    type: 'FID',
                    value: entry.processingStart - entry.startTime
                });
            }
        }).observe({ type: 'first-input', buffered: true });
        
        // 监控 JS 错误
        window.addEventListener('error', (event) => {
            this.report({
                type: 'JS_ERROR',
                message: event.message,
                filename: event.filename,
                lineno: event.lineno
            });
        });
        
        // 监控 Promise 错误
        window.addEventListener('unhandledrejection', (event) => {
            this.report({
                type: 'PROMISE_ERROR',
                reason: event.reason?.message
            });
        });
    }
}
```
