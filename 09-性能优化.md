## 9.1 前端性能指标

### 9.1.1 Core Web Vitals

#### 知识点详解

Google 定义的核心网页指标，直接影响 SEO 排名。

**LCP（Largest Contentful Paint）- 最大内容绘制：**

- 衡量页面主要内容的加载速度
- 测量视口内最大元素（图片、视频、文本块）的渲染时间
- 目标：< 2.5 秒

```javascript
// 监测 LCP
new PerformanceObserver((entryList) => {
    const entries = entryList.getEntries();
    const lastEntry = entries[entries.length - 1];
    console.log('LCP:', lastEntry.startTime);
}).observe({ type: 'largest-contentful-paint', buffered: true });
```

**FID（First Input Delay）/ INP（Interaction to Next Paint）：**

- FID：首次输入延迟，用户首次交互到浏览器响应的时间
- INP（新指标）：所有交互的响应延迟
- 目标：FID < 100ms，INP < 200ms

```javascript
// 监测 FID
new PerformanceObserver((entryList) => {
    for (const entry of entryList.getEntries()) {
        console.log('FID:', entry.processingStart - entry.startTime);
    }
}).observe({ type: 'first-input', buffered: true });
```

**CLS（Cumulative Layout Shift）- 累积布局偏移：**

- 衡量页面视觉稳定性
- 测量意外布局偏移的总量
- 目标：< 0.1

```javascript
// 监测 CLS
let clsValue = 0;
new PerformanceObserver((entryList) => {
    for (const entry of entryList.getEntries()) {
        if (!entry.hadRecentInput) {
            clsValue += entry.value;
        }
    }
    console.log('CLS:', clsValue);
}).observe({ type: 'layout-shift', buffered: true });
```

**FCP（First Contentful Paint）- 首次内容绘制：**

- 浏览器渲染第一个 DOM 内容的时间
- 目标：< 1.8 秒

**TTI（Time to Interactive）- 可交互时间：**

- 页面完全可交互的时间
- 目标：< 3.8 秒

**使用 web-vitals 库：**

```javascript
import { getCLS, getFID, getFCP, getLCP, getTTFB } from 'web-vitals';

function sendToAnalytics(metric) {
    const body = JSON.stringify(metric);
    navigator.sendBeacon('/analytics', body);
}

getCLS(sendToAnalytics);
getFID(sendToAnalytics);
getFCP(sendToAnalytics);
getLCP(sendToAnalytics);
getTTFB(sendToAnalytics);
```

**减少重排重绘：**

```javascript
// 1. 批量 DOM 操作
// 不好：多次操作 DOM
for (let i = 0; i < 1000; i++) {
    document.body.appendChild(document.createElement('div'));
}

// 好：使用 DocumentFragment
const fragment = document.createDocumentFragment();
for (let i = 0; i < 1000; i++) {
    fragment.appendChild(document.createElement('div'));
}
document.body.appendChild(fragment);

// 2. 避免强制同步布局
// 不好：读写交替
for (let i = 0; i < elements.length; i++) {
    elements[i].style.width = elements[i].offsetWidth + 10 + 'px';
}

// 好：先读后写
const widths = elements.map(el => el.offsetWidth);
elements.forEach((el, i) => {
    el.style.width = widths[i] + 10 + 'px';
});

// 3. 使用 CSS transform 代替 top/left
// 不好：触发重排
element.style.top = '100px';

// 好：只触发合成
element.style.transform = 'translateY(100px)';

// 4. 使用 will-change 提示浏览器
.animated {
    will-change: transform;
}

// 5. 使用 requestAnimationFrame
function animate() {
    element.style.transform = `translateX(${x}px)`;
    x++;
    requestAnimationFrame(animate);
}
requestAnimationFrame(animate);
```

### 9.2.3 浏览器缓存策略

```javascript
// Service Worker 缓存
// sw.js
const CACHE_NAME = 'v1';
const STATIC_ASSETS = [
    '/',
    '/index.html',
    '/styles.css',
    '/app.js'
];

// 安装：缓存静态资源
self.addEventListener('install', (event) => {
    event.waitUntil(
        caches.open(CACHE_NAME).then(cache => {
            return cache.addAll(STATIC_ASSETS);
        })
    );
});

// 激活：清理旧缓存
self.addEventListener('activate', (event) => {
    event.waitUntil(
        caches.keys().then(keys => {
            return Promise.all(
                keys.filter(key => key !== CACHE_NAME)
                    .map(key => caches.delete(key))
            );
        })
    );
});

// 拦截请求
self.addEventListener('fetch', (event) => {
    event.respondWith(
        caches.match(event.request).then(response => {
            // 缓存优先策略
            if (response) return response;
            
            // 网络请求
            return fetch(event.request).then(response => {
                const clone = response.clone();
                caches.open(CACHE_NAME).then(cache => {
                    cache.put(event.request, clone);
                });
                return response;
            });
        })
    );
});
```

```jsx
                            style={{ height: itemHeight }}

                            {item.name}
                        </div>
```

### 9.2.5 图片懒加载

```javascript
// 方式1：Intersection Observer API（推荐）
function lazyLoadImages() {
    const images = document.querySelectorAll('img[data-src]');
    
    const observer = new IntersectionObserver((entries) => {
        entries.forEach(entry => {
            if (entry.isIntersecting) {
                const img = entry.target;
                img.src = img.dataset.src;
                img.removeAttribute('data-src');
                observer.unobserve(img);
            }
        });
    }, {
        rootMargin: '200px 0px'  // 提前 200px 加载
    });
    
    images.forEach(img => observer.observe(img));
}

// 方式2：HTML 原生 loading 属性
<img src="image.jpg" loading="lazy" alt="...">

// 方式3：滚动事件（旧方式）
function lazyLoad() {
    const images = document.querySelectorAll('img[data-src]');
    
    images.forEach(img => {
        const rect = img.getBoundingClientRect();
        if (rect.top < window.innerHeight + 200) {
            img.src = img.dataset.src;
            img.removeAttribute('data-src');
        }
    });
}

window.addEventListener('scroll', throttle(lazyLoad, 100));
```

### 9.x.x Web 应用中静态资源加载失败的降级处理 [重要]

#### 知识点详解

静态资源（JS、CSS、图片、字体等）加载失败时，需要优雅降级，保证页面核心功能可用。

**1. 图片加载失败降级：**

```html

<img 
  src="https://cdn.example.com/photo.jpg" 
  alt="用户头像"
  onerror="this.src='/assets/default-avatar.png'; this.onerror=null;"
/>

<!-- React 组件 -->
function LazyImage({ src, fallback = '/default.png', alt }) {
  const [imgSrc, setImgSrc] = useState(src);
  const [error, setError] = useState(false);

  return (
    <img
      src={error ? fallback : imgSrc}
      alt={alt}
      onError={() => setError(true)}
    />
  );
}

// Vue 指令封装
const imageError = {
  mounted(el, binding) {
    el.__handleError__ = () => {
      el.src = binding.value || '/assets/default.png';
    };
    el.addEventListener('error', el.__handleError__);
  },
  unmounted(el) {
    el.removeEventListener('error', el.__handleError__);

```

```javascript
// 多 CDN 降级策略
const CDN_LIST = [
  'https://cdn1.example.com/',
  'https://cdn2.example.com/',
  'https://cdn3.example.com/'
];

async function loadScript(src) {
  for (const cdn of CDN_LIST) {
    const url = cdn + src;
    try {
      await loadScriptPromise(url);
      console.log('成功加载:', url);
      return;
    } catch (e) {
      console.warn(`CDN ${cdn} 失败，尝试下一个`);
    }

function loadScriptPromise(url) {
  return new Promise((resolve, reject) => {
    const script = document.createElement('script');
    script.src = url;
    script.onload = resolve;
    script.onerror = reject;
    document.head.appendChild(script);
  });
}
```

### 9.x.x 前端水印的实现方案 [重要]

#### 知识点详解

前端水印用于防止截图泄密，常用于数据平台、OA 系统等场景。

**1. Canvas 生成图片水印（最常见）：**

```javascript
function createWatermark(text = '水印文字', options = {}) {
  const {
    width = 200,
    height = 100,
    font = '16px Microsoft YaHei',
    color = 'rgba(0, 0, 0, 0.1)',
    angle = -30
  } = options;

  const canvas = document.createElement('canvas');
  canvas.width = width;
  canvas.height = height;
  const ctx = canvas.getContext('2d');

  ctx.font = font;
  ctx.fillStyle = color;
  ctx.textAlign = 'center';
  ctx.textBaseline = 'middle';

  ctx.translate(width / 2, height / 2);
  ctx.rotate(angle * Math.PI / 180);
  ctx.fillText(text, 0, 0);

  return canvas.toDataURL('image/png');
}

// 将水印应用到页面
function applyWatermark(container, watermarkUrl) {
  const watermark = document.createElement('div');
  watermark.style.cssText = `
    position: absolute;
    top: 0; left: 0; right: 0; bottom: 0;
    background-image: url(${watermarkUrl});
    background-repeat: repeat;
    pointer-events: none;
    z-index: 9999;
    opacity: 1;
  `;
  // 插入到容器（覆盖内容但不影响交互）
  container.style.position = 'relative';
  container.appendChild(watermark);
}
```

## 9.9 requestIdleCallback API

#### 知识点详解

**什么是 requestIdleCallback？**

浏览器提供的 API，用于在浏览器空闲时执行低优先级任务。

```javascript
// 基本用法
requestIdleCallback((deadline) => {
    // deadline.timeRemaining() 返回剩余时间
    console.log('执行任务');
});

// 带超时
requestIdleCallback((deadline) => {
    // 最多等待 2000ms
}, { timeout: 2000 });
```

**Deadline 对象：**

```javascript
requestIdleCallback((deadline) => {
    // 剩余可用时间（毫秒）
    console.log(deadline.timeRemaining());
    
    // 是否超时
    console.log(deadline.didTimeout);
});
```

**使用场景：**

```javascript
// 1. 批量处理数据
function processLargeArray(data) {
    let index = 0;
    
    function processChunk(deadline) {
        while (index < data.length && deadline.timeRemaining() > 1) {
            process(data[index]);
            index++;
        }
        
        if (index < data.length) {
            requestIdleCallback(processChunk);
        }
    }
    
    requestIdleCallback(processChunk);
}

// 2. 延迟非关键渲染
function deferNonCriticalRender() {
    requestIdleCallback(() => {
        sendAnalytics();
    });
}
```

**Polyfill：**

```javascript
window.requestIdleCallback = window.requestIdleCallback || function(cb) {
    return setTimeout(() => {
        cb({
            timeRemaining: () => 50,
            didTimeout: false
        });
    }, 1);
};
```

#### 真实面试题

**题目：requestIdleCallback 是什么？适用于哪些场景？**

**满分答案：**

**定义：** 浏览器提供的 API，在页面空闲时执行非关键任务。

**适用场景：**
1. 大量数据处理（分批执行）
2. 非关键渲染延迟
3. 埋点/统计上报
4. 预加载非关键资源

**与 setTimeout 对比：**
- setTimeout：指定延迟后执行，无法控制优先级
- requestIdleCallback：浏览器空闲时执行，低优先级，更节能

## 9.11 大量任务处理：保证浏览器不卡顿

#### 知识点详解

**核心思路：分批处理 + 空闲执行**

**1. 数组分块（Chunking）：**

```javascript
function processItems(items, chunkSize = 1000) {
    let index = 0;
    
    function processChunk() {
        const chunk = items.slice(index, index + chunkSize);
        
        chunk.forEach(item => doSomething(item));
        
        index += chunkSize;
        
        if (index < items.length) {
            setTimeout(processChunk, 0);
        }
    }
    
    processChunk();
}
```

**2. 使用 requestIdleCallback：**

```javascript
function processTasks(tasks) {
    let index = 0;
    
    function processNext(deadline) {
        while (index < tasks.length && deadline.timeRemaining() > 1) {
            doSomething(tasks[index]);
            index++;
        }
        
        if (index < tasks.length) {
            requestIdleCallback(processNext);
        }
    }
    
    requestIdleCallback(processNext);
}
```

**3. 使用 Web Worker：**

```javascript

worker.postMessage({ tasks: hugeArray });
worker.onmessage = (e) => console.log('处理完成', e.data);

```

## 9.12 虚拟滚动加载原理

#### 知识点详解

**核心原理：只渲染可视区域内的列表项**

```
┌─────────────────────┐
│   可视区域 viewport │  ← 只渲染这里
├─────────────────────┤
│  [item 0]           │
│  [item 1]           │
│  [item 2]           │
├─────────────────────┤
│  (item 5 ~ 9995)    │  ← 不渲染
└─────────────────────┘
```

**关键计算：**

```javascript
// 1. 计算可视区域能显示多少项
const visibleCount = Math.ceil(viewportHeight / itemHeight);

// 2. 计算起始索引
const startIndex = Math.floor(scrollTop / itemHeight);

// 3. 计算结束索引
const endIndex = startIndex + visibleCount;

// 4. 计算偏移量（保持滚动位置）
const offsetY = startIndex * itemHeight;
```

**简单实现：**

```javascript
class VirtualList {
    constructor(options) {
        this.container = options.container;
        this.itemHeight = options.itemHeight;
        this.items = options.items;
        this.renderItem = options.renderItem;
        
        this.viewportHeight = this.container.clientHeight;
        this.container.addEventListener('scroll', () => this.render());
        this.render();
    }
    
    getVisibleRange() {
        const scrollTop = this.container.scrollTop;
        const startIndex = Math.floor(scrollTop / this.itemHeight);
        const visibleCount = Math.ceil(this.viewportHeight / this.itemHeight);
        const buffer = 5;
        
        return {
            start: Math.max(0, startIndex - buffer),
            end: Math.min(this.items.length, startIndex + visibleCount + buffer)
        };
    }
    
    render() {
        const { start, end } = this.getVisibleRange();
        
        // 总高度（占位）
        const totalHeight = this.items.length * this.itemHeight;
        this.container.style.height = totalHeight + 'px';
        
        // 渲染可见项
        const visibleItems = this.items.slice(start, end);
        const html = visibleItems.map((item, index) => {
            const actualIndex = start + index;
            const top = actualIndex * this.itemHeight;
            return `<div style="position:absolute;top:${top}px;height:${this.itemHeight}px">
                ${this.renderItem(item, actualIndex)}
            </div>`;
        }).join('');
        
        this.container.innerHTML = html;
    }
}
```

#### 真实面试题

**题目：虚拟滚动的原理是什么？如何实现一个简单的虚拟列表？**

**满分答案：**

**核心原理：**
1. 计算可视区域能显示的元素数量
2. 根据滚动位置计算起始索引
3. 只渲染可视区域内的元素
4. 用占位元素保持滚动条高度

1. 固定高度 vs 动态高度

## 9.14 Web Workers 处理视频帧数据

#### 知识点详解

```
主线程:
[处理视频帧] → [UI冻结] → [用户无法交互]
         ↓
使用 Worker:
主线程: [发送帧数据] → [UI响应]
Worker:  [处理帧数据] → [返回结果]
```

**Web Worker 处理视频帧：**

```javascript

// 2. 主线程：提取视频帧并发送到 Worker
const video = document.getElementById('video');
const canvas = document.createElement('canvas');
const ctx = canvas.getContext('2d');

function extractFrame(time) {
    video.currentTime = time;
    
    video.onseeked = () => {
        canvas.width = video.videoWidth;
        canvas.height = video.videoHeight;
        ctx.drawImage(video, 0, 0);
        
        // 获取图像数据
        const imageData = ctx.getImageData(0, 0, canvas.width, canvas.height);
        
        // 发送到 Worker 处理
        worker.postMessage({
            type: 'processFrame',
            imageData: imageData.data,
            width: canvas.width,
            height: canvas.height
        }, [imageData.data.buffer]);  // 使用 Transferable 对象
    };
}

// 3. Worker: 处理帧数据 (videoWorker.js)
self.onmessage = function(e) {
    const { type, imageData, width, height } = e.data;
    
    if (type === 'processFrame') {
        // 图像处理：例如灰度转换
        const result = processImage(imageData, width, height);
        
        // 返回处理结果
        self.postMessage({
            type: 'frameProcessed',
            data: result
        }, [result.buffer]);
    }
};

function processImage(data, width, height) {
    const output = new Uint8ClampedArray(data.length);
    
    for (let i = 0; i < data.length; i += 4) {
        // 灰度转换
        const gray = data[i] * 0.299 + data[i + 1] * 0.587 + data[i + 2] * 0.114;
        output[i] = gray;     // R
        output[i + 1] = gray; // G
        output[i + 2] = gray; // B
        output[i + 3] = data[i + 3]; // A
    }
    
    return output;
}

// 4. 接收处理结果
worker.onmessage = function(e) {
    if (e.data.type === 'frameProcessed') {
        // 将处理后的数据显示到 Canvas
        const processedData = new ImageData(
            new Uint8ClampedArray(e.data.data),
            width,
            height
        );
        ctx.putImageData(processedData, 0, 0);
    }
};
```

## 9.15 大文件上传（4K视频）整体流程

#### 知识点详解

**大文件上传核心思路：分片 + 断点续传**

```
文件切片:
[chunk1][chunk2][chunk3][chunk4][chunk5]...
   ↓        ↓       ↓       ↓       ↓
  上传    上传    上传    上传    上传
```

**完整实现：**

```javascript
// 1. 分片工具
function sliceFile(file, chunkSize = 2 * 1024 * 1024) {
    const chunks = [];
    let start = 0;
    
    while (start < file.size) {
        const chunk = file.slice(start, start + chunkSize);
        chunks.push({
            chunk: chunk,
            start: start,
            end: Math.min(start + chunkSize, file.size),
            index: chunks.length,
            filename: file.name,
            totalSize: file.size
        });
        start += chunkSize;
    }
    
    return chunks;
}

// 2. 计算文件 hash（用于秒传和断点续传）
async function calculateFileHash(file) {
    const worker = new Worker('hashWorker.js');
    
    return new Promise((resolve) => {
        worker.postMessage(file);
        worker.onmessage = (e) => {
            resolve(e.data.hash);
            worker.terminate();
        };
    });
}

// 3. 上传单个分片
async function uploadChunk(chunk, uploadId) {
    const formData = new FormData();
    formData.append('chunk', chunk.chunk);
    formData.append('index', chunk.index);
    formData.append('filename', chunk.filename);
    formData.append('uploadId', uploadId);
    
    return fetch('/api/upload/chunk', {
        method: 'POST',
        body: formData
    });
}

// 4. 完整上传流程
class LargeFileUploader {
    constructor(options) {
        this.file = options.file;
        this.chunkSize = options.chunkSize || 2 * 1024 * 1024;
        this.concurrency = options.concurrency || 3;
        this.onProgress = options.onProgress || (() => {});
    }
    
    async upload() {
        // 1. 计算文件 hash
        const fileHash = await calculateFileHash(this.file);
        
        // 2. 检查是否存在（秒传）
        const checkResult = await this.checkExist(fileHash);
        if (checkResult.exists) {
            return { success: true, message: '秒传成功' };
        }
        
        // 3. 获取已上传的分片
        const uploadedChunks = await this.getUploadedChunks(fileHash);
        
        // 4. 分片上传
        const chunks = sliceFile(this.file, this.chunkSize);
        const pendingChunks = chunks.filter(c => !uploadedChunks.includes(c.index));
        
        // 并发控制上传
        await this.uploadWithConcurrency(pendingChunks, fileHash);
        
        // 5. 通知服务端合并
        await this.mergeChunks(fileHash);
        
        return { success: true };
    }
    
    async uploadWithConcurrency(chunks, fileHash) {
        let index = 0;
        
        const run = async () => {
            while (index < chunks.length) {
                const chunk = chunks[index++];
                await this.uploadSingleChunk(chunk, fileHash);
                this.onProgress(index / chunks.length);
            }
        };
        
        // 启动并发上传
        const workers = Array(this.concurrency).fill(null).map(run);
        await Promise.all(workers);
    }
    
    async checkExist(hash) {
        const res = await fetch(`/api/upload/check?hash=${hash}`);
        return res.json();
    }
    
    async getUploadedChunks(hash) {
        const res = await fetch(`/api/upload/status?hash=${hash}`);
        const data = await res.json();
        return data.uploadedChunks || [];
    }
    
    async uploadSingleChunk(chunk, fileHash) {
        const formData = new FormData();
        formData.append('chunk', chunk.chunk);
        formData.append('index', chunk.index);
        formData.append('hash', fileHash);
        
        await fetch('/api/upload/chunk', {
            method: 'POST',
            body: formData
        });
    }
    
    async mergeChunks(fileHash) {
        await fetch('/api/upload/merge', {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify({ hash: fileHash, filename: this.file.name })
        });
    }
}

// 使用
const uploader = new LargeFileUploader({
    file: videoFile,
    chunkSize: 2 * 1024 * 1024,  // 2MB
    concurrency: 4,
    onProgress: (percent) => {
        console.log(`上传进度: ${Math.round(percent * 100)}%`);
    }
});

uploader.upload().then(result => {
    console.log('上传完成', result);
});
```

```javascript
const fs = require('fs');
const path = require('path');

async function mergeChunks(req, res) {
    const { hash, filename } = req.body;
    const tempDir = path.join(__dirname, 'uploads', hash);
    const outputPath = path.join(__dirname, 'uploads', filename);
    
    const writeStream = fs.createWriteStream(outputPath);
    
    // 按序号读取并合并
    const files = fs.readdirSync(tempDir).sort((a, b) => {
        return parseInt(a) - parseInt(b);
    });
    
    for (const file of files) {
        const chunkPath = path.join(tempDir, file);
        const chunkData = fs.readFileSync(chunkPath);
        writeStream.write(chunkData);
        fs.unlinkSync(chunkPath);  // 删除临时分片
    }
    
    writeStream.end();
    fs.rmdirSync(tempDir);  // 删除临时目录
    
    res.json({ success: true, url: `/uploads/${filename}` });
}
```

## 9.17 万级数据长列表渲染优化

#### 知识点详解

**问题根源：**

```
10000 个 DOM 节点 → 布局计算量巨大 → 滚动卡顿
```

**方案一：虚拟滚动（Virtual Scroll）— 最优解**

```javascript
// 核心思路：只渲染可视区域内的元素
class VirtualList {
    constructor(container, options) {
        this.container = container;
        this.itemHeight = options.itemHeight;  // 每项高度（固定）
        this.data = options.data;
        this.bufferSize = options.bufferSize || 5;  // 缓冲区

        this.containerHeight = container.clientHeight;
        this.visibleCount = Math.ceil(this.containerHeight / this.itemHeight);

        this.scrollTop = 0;
        this.startIndex = 0;

        this._init();
    }

    _init() {
        // 撑开滚动高度
        this.phantom = document.createElement('div');
        this.phantom.style.height = `${this.data.length * this.itemHeight}px`;

        // 实际渲染容器
        this.content = document.createElement('div');
        this.content.style.position = 'absolute';
        this.content.style.top = '0';
        this.content.style.width = '100%';

        this.container.style.position = 'relative';
        this.container.style.overflow = 'auto';
        this.container.appendChild(this.phantom);
        this.container.appendChild(this.content);

        this.container.addEventListener('scroll', this._onScroll.bind(this));
        this._render();
    }

    _onScroll() {
        this.scrollTop = this.container.scrollTop;
        this.startIndex = Math.floor(this.scrollTop / this.itemHeight);
        this._render();
    }

    _render() {
        const start = Math.max(0, this.startIndex - this.bufferSize);
        const end = Math.min(
            this.data.length,
            this.startIndex + this.visibleCount + this.bufferSize
        );

        // 偏移量
        const offsetY = start * this.itemHeight;
        this.content.style.transform = `translateY(${offsetY}px)`;

        // 只渲染可见 + 缓冲区的数据
        const visibleData = this.data.slice(start, end);
        this.content.innerHTML = visibleData
            .map(item => `<div style="height:${this.itemHeight}px">${item.name}</div>`)
            .join('');
    }
}

// 使用
const list = new VirtualList(document.getElementById('container'), {
    itemHeight: 50,
    data: Array.from({ length: 10000 }, (_, i) => ({ id: i, name: `Item ${i}` })),
    bufferSize: 5
});
```

```javascript
// 分批渲染，不阻塞主线程
function renderInChunks(data, chunkSize = 100) {
    let index = 0;

    function renderChunk(deadline) {
        while (index < data.length && deadline.timeRemaining() > 0) {
            const end = Math.min(index + chunkSize, data.length);
            const fragment = document.createDocumentFragment();

            for (let i = index; i < end; i++) {
                const el = document.createElement('div');
                el.textContent = data[i].name;
                fragment.appendChild(el);

            document.getElementById('list').appendChild(fragment);
            index = end;
        }

        if (index < data.length) {
            requestIdleCallback(renderChunk);
        }
    }

    requestIdleCallback(renderChunk);
}
```

```javascript
// ✅ 推荐方案：分批次更新
function OptimizedTypewriter({ stream }) {
    const [text, setText] = useState('');
    const bufferRef = useRef('');
    const rafRef = useRef(null);

    useEffect(() => {
        const processBuffer = () => {
            if (bufferRef.current) {
                setText(prev => prev + bufferRef.current);
                bufferRef.current = '';
            }
            rafRef.current = requestAnimationFrame(processBuffer);
        };

        // 1. 收集字符到 buffer
        stream.onChar(char => {
            bufferRef.current += char;
        });

        // 2. 每帧最多更新一次（~16ms）
        rafRef.current = requestAnimationFrame(processBuffer);

        return () => {
            if (rafRef.current) cancelAnimationFrame(rafRef.current);
        };
    }, [stream]);

    return <div>{text}</div>;
}
```

```javascript
            if (bufferRef.current) {
                containerRef.current.textContent += bufferRef.current;
                bufferRef.current = '';
            }
            requestAnimationFrame(process);
        };
        requestAnimationFrame(process);

        stream.onChar(char => { bufferRef.current += char; });
    }, []);

    return <div ref={containerRef} />;
}
```

**AI 前端的三大 Worker 场景：**

**1. Token 计数（避免主线程阻塞）：**

```javascript
// AI API 通常按 Token 收费，发送前需要估算 Token 数

const tokenizerWorker = new Worker('tokenizer.js');

    console.log('Token 数:', e.data.count);

        showWarning('超出上下文限制');
    }
};
```

**2. 音频转录（Whisper Web）：**

```javascript
// 实时语音转文字，跑在 Worker 中避免卡顿
const whisperWorker = new Worker('whisper.worker.js');

whisperWorker.onmessage = (e) => {
    const { text, isFinal } = e.data;
    if (isFinal) {
        // 最终结果，发送给 AI
        sendToAI(text);
    } else {
        // 实时转录，显示在输入框
        updateInputText(text);
    }
};

// 音频数据传入 Worker
audioStream.getTracks().forEach(track => {
    const audioWorklet = new AudioWorkletNode(ctx, 'whisper-processor');
    audioWorklet.port.postMessage({ audioData });
});
```

**3. 大文本处理：**

```javascript
// AI 输出的长文本在 Worker 中做格式转换
const textProcessorWorker = new Worker('textProcessor.js');

textProcessorWorker.postMessage({
    rawText: largeText,
```

## 9.21 TTFT 超过 5 秒的用户体验优化

### 知识点详解

**TTFT（Time To First Token）**：用户发起请求到收到第一个 Token 的时间，是 AI 对话体验的核心指标。

**前端能做的优化（不等后端优化）：**

```typescript
// 1. 阶段状态提示
const getStatusLabel = (phase: 'connecting' | 'thinking' | 'streaming') => {
    if (phase === 'connecting') return '正在建立连接...';
    if (phase === 'thinking') return 'AI 正在思考...';
    return '正在生成回复...';
};

// 2. 倒计时超时兜底
const TIMEOUT_SECONDS = 30;
useEffect(() => {
    if (!isLoading) { setElapsed(0); return; }
    const timer = setInterval(() => {
        setElapsed(e => {
            if (e >= TIMEOUT_SECONDS) {
                onTimeout();
                return e;
            }
            return e + 1;
        });
    }, 1000);
    return () => clearInterval(timer);
}, [isLoading]);
```

### 真实面试题

**题目：当 AI 接口响应非常慢（TTFT 超过 5 秒）时，你会通过哪些手段优化用户体验？**

**满分答案：**

**分层策略（从感知快到实际优化）：**

| 手段 | 效果 | 优先级 |
|------|------|--------|
| 阶段状态提示 | 告知用户 AI 在工作 | ⭐⭐⭐ |
| 思考动画 | 比 spinner 更生动 | ⭐⭐⭐ |
| 取消/重试按钮 | 给用户控制感 | ⭐⭐⭐⭐ |
| 倒计时超时兜底 | 防止无限等待 | ⭐⭐⭐⭐ |
| 预热连接 | 减少连接耗时 | ⭐⭐ |
| 相关推荐 | 让他有事情做 | ⭐⭐⭐ |

**具体实现：**

```tsx
// 1. 思考状态动画
<div className="thinking">
    <div className="dots">
        <span /><span /><span />
    </div>
    <span>AI 正在思考中... {elapsed}s</span>
</div>

// 2. 相关推荐（让用户不等）
{isSlow && (
    <div className="suggestions">
        <p>你可能想了解：</p>
        <button onClick={() => setInput('如何优化性能？')}>如何优化性能？</button>
        <button onClick={() => setInput('最新版本有哪些特性？')}>最新版本有哪些特性？</button>
    </div>
)}

// 3. 取消按钮
<button onClick={cancel} disabled={isComplete}>
    {isComplete ? '已生成' : elapsed > TIMEOUT ? '超时，重试' : '取消'}
</button>
```

**最佳实践：**
- TTFT < 2s：直接显示流式输出
- TTFT 2-5s：显示连接 + 思考状态
- TTFT > 5s：思考动画 + 相关推荐 + 倒计时
- TTFT > 30s：超时提示 + 自动取消
